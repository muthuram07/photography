# Photography Portfolio

This is a stunning 3D photography portfolio built with Next.js and React Three Fiber. It showcases a unique and immersive way to display photographic art.

## Features

*   **Immersive 3D gallery experience:** Images are displayed in a 3D space, providing a unique viewing experience.
*   **Interactive navigation:** Users can navigate through the gallery using the mouse wheel, arrow keys, or touch gestures.
*   **Auto-play:** The gallery can automatically scroll through the images, with the auto-play feature resuming after a period of inactivity.
*   **Dynamic image loading:** Images are loaded dynamically from a Cloudinary portfolio.
*   **Stunning visual effects:** The gallery includes blur-in and blur-out effects for smooth transitions between images.
*   **Responsive design:** The gallery is designed to work on a variety of screen sizes, from mobile devices to desktops.

## Tech Stack

*   **Next.js:** A React framework for building server-rendered and statically-generated web applications.
*   **React Three Fiber:** A React renderer for Three.js, used to create the 3D gallery.
*   **React Three Drei:** A collection of useful helpers and abstractions for React Three Fiber.
*   **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.
*   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.
*   **Cloudinary:** A cloud-based image and video management service.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js (v20 or later)
*   npm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/your_project_name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Running the Development Server

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
